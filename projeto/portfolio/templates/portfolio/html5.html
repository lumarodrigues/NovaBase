HTML5 (Hypertext Markup Language)





Descrição: O HTML é a linguagem base da internet, o HTML5 é a quinta versão da linguagem. Esta nova versão traz consigo importantes mudanças quanto ao papel do HTML no mundo da Web, através de novas funcionalidades como semântica e acessibilidade.



Objetivo do curso: Preparar o aluno iniciante na utilização da linguagem HTML. Como a utilização da internet cresce a cada dia, o aprendizado de HTML é essencial para o aluno ingressar no mercado altamente promissor de desenvolvimento web.



Pré-requisito: Windows

Carga horária: 24 horas

Modalidade: Presencial

Versões: HTML5

Categoria: Web Design
