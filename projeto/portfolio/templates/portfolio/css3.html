CSS3 (Cascading Style Sheets)





Descrição: O CSS promove a separação entre o código (HTML) e o estilo (formatação) das páginas de internet. Através do uso de CSS, conseguimos efeitos de formatação e de layouts muito atraentes para páginas de internet, além da facilidade de criação de modelos (temas) para páginas de internet.



Objetivo do curso: O curso é voltado para o aluno que já conhece HTML e que está se desenvolvendo na área de desenvolvimento web. O objetivo é ensinar o conceito do funcionamento do CSS para que o aluno aplique seus conhecimentos, deixando suas páginas com formato de trabalho profissional.



Pré-requisito: Conhecimento em HTML

Carga horária: 16 horas

Modalidade: Presencial

Versões: HTML5

Categoria: Web Design



O que o aluno aprenderá?

● Introdução ao HTML

●Iniciar o uso de CSS em páginas HTML

●Formatar o fundo das páginas (background)

●Formatação de margens

●Formatação de textos

●Formatação de parágrafos

●Propriedades de posicionamento de objetos

●Posicionar objetos (frente e atrás)

●Bordas em objetos

●Listas ordenadas e não ordenadas com CSS

●Formatar formulários
