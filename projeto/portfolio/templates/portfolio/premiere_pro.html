Premiere Pro



Descrição: O Premiere é um dos principais softwares de edição de vídeo para cinema, TV e a Web, dentro da Web ele é bastante usado na edição de vídeos para o YouTube.



Objetivo do curso: Nesse curso o aluno trabalhará aprenderá o básico para a edição de vídeos, passando pelos diferentes tipos de corte, ajustes de som, aplicação de efeitos e exportação do seu projeto.



Pré-requisito: Windows

Carga horária: 24 horas

Modalidade: Presencial e EAD

Versões: Premiere Pro CC 2019



O que o aluno aprenderá?

● Criar e configurar uma área de trabalho

● Conhecer os diferentes painéis do programa

● Criar um projeto

● Conhecer os diferentes formatos de vídeo

● Importar e organizar arquivos para o projeto

● Entender a linha do tempo e seu empilhamento

● Criar animação com caracteres

● Aplicar transições

● Criar cortes secos e cortes suaves

● Aplicar efeitos variados

● Utilizar os modos de mesclagem

● Simular um vídeo antigo

● Utilizar as predefinições de cores

● Mesclar imagens

● Mixar músicas para efeitos de transição

● Conhecer o conceito de timelapse

● Exportar seu projeto
