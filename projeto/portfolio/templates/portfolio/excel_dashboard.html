Excel - Dashboard



Descrição: O Excel é um programa de elaboração de planilhas, devido a sua flexibilidade é utilizado em várias aplicações como planilhas administrativas, comerciais, controle de estoques, entre outras, disponibilizando dados que servem para tomadas de decisões.



Objetivo do curso: Fazer com o que o aluno que já utiliza o Excel possa se aprofundar em sua utilização, aprendendo recursos avançados que irão possibilitar o controle de dados através da elaboração de planilhas mais complexas.



Pré-requisito: Windows

Carga horária: 16 horas

Modalidade: Presencial e EAD

Versões: 365 (Compatível com 2019) e 2016



O que o aluno aprenderá?

● Ferramenta letreiro

● Criar um modelo de dados

● Selecionar os dados

● Usar KPIs

● Criar um dashboard completo

● Usar minigráficos

● Gráfico de termômetro

● Usar botões de opção e macro

● Usar caixas de combinação
