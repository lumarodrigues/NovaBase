Lógica de Programação



Descrição: Para o aluno que deseja se tornar programador, o curso de lógica é o primeiro passo, pois, independente da linguagem a ser aprendida, será necessário utilizar e dominar os conceitos básicos de lógica de programação.



Objetivo do curso: O curso de Lógica de Programação irá preparar o aluno para entender os princípios básicos de uma linguagem de programação, para que tenha toda a base necessária iniciar na área de programação.



Pré-requisito: Windows

Carga horária: 12 horas

Modalidade: Presencial



O que o aluno aprenderá?

● Introdução à lógica de programação

● Escrever algoritmos

● Estruturas de decisão

● Operadores lógicos

● Estruturas de repetição
