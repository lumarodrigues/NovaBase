Revit - Base



Descrição: O Revit é um software para a arquitetura, ele utiliza o conceito de modelagem “BIM”, esse conceito oferece uma associatividade bi-direcional, o que permite que uma mudança feita em qualquer lugar do projeto refletirá por todo o projeto.



Objetivo do curso: Neste curso você criará um projeto completo, desde a planta até a renderização da casa com mobília e iluminação.



Pré-requisito: Windows

Carga horária: 36 horas

Modalidade: Presencial

Versões: Revit 2018



O que o aluno aprenderá?

● Criar esboços e peças 3D

● Iniciar um novo projeto

● Ajustar a altura dos níveis

● Criar paredes retas e curvas

● Utilizar cotas para conferência

● Criar rodapés e molduras de gesso

● Importar plantas do AutoCAD

● Inserir portas e janelas no projeto

● Inserir cotas

● Identificar ambientes

● Criar pisos

● Inserir soleiras e peitoris

● Inserir o mobiliário

● Inserir forro

● Inserir iluminação

● Inserir ventilador de teto

● Renderizar o projeto
