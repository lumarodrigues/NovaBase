Illustrator



Descrição: O Illustrator é um editor de imagens vetoriais desenvolvido e comercializado pela Adobe Systems. Foi criado inicialmente para o Apple Macintosh em 1985 como complemento comercial de software de fontes da Adobe e da tecnologia PostScript desenvolvida pela empresa.



Objetivo do curso: Ensinar ao aluno o básico do Illustrator, com conceitos de uso das principais ferramentas de desenho, preenchimento e uso de bibliotecas, fornecendo condições para o contínuo desenvolvimento do aluno.



Pré-requisito: Windows

Carga horária: 24 horas

Modalidade: Presencial

Versões: Illustrator CC 2015



O que o aluno aprenderá?

● Trabalhar com preenchimento

● Trabalhar com texto

● Trabalhar com vetores

● Interação entre objetos vetoriais

● Trabalhar com camadas

● Vetorizar

● Utilizar a ferramenta “Caneta”

● Utilizar símbolos

● Aplicar estilos gráficos e utilizar pincéis

● Utilizar a ferramenta “Malha”

● Utilizar a ferramenta de perspectiva

● Preparar arquivos para impressão
